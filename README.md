# MYTO Web
Sitio web oficial de Terapias MYTO - Tucumán, Argentina.
## Sobre MYTO
Centro especializado en terapias orientales y artes marciales. Servicios:
- **Terapias:** Masaje Oriental, Ventosas (Cupping), Acupuntura, Reiki
- **Cursos:** Arte Marcial, Tai Chi, Yoga, Defensa Personal
## 🛠️ Stack Tecnológico
- **Framework:** React 19
- **Lenguaje:** TypeScript
- **Build Tool:** Vite
- **Routing:** React Router 7
- **Icons:** React Icons
## 🚀 Getting Started
### Prerrequisitos
- Node.js 18+
- npm 9+
### Instalación
```bash
npm install
Desarrollo
npm run dev
Build para producción
npm run build
📁 Estructura
src/
├── components/     # Componentes reutilizables
├── pages/         # Páginas principales
├── App.tsx        # Componente principal
├── main.tsx       # Entry point
└── router.tsx     # Configuración de rutas
